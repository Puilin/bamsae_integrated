## 기획
### 시나리오
    A사와 B사는 멀티플렉스 영화관을 운영중임. 이듬해 A사가 B사를 인수 통합함에 따라
    두 회사가 관리하던 영화관 관리 시스템을 통합 또는 재구축해야 하는 상황임.
    플랫폼 솔루션 제공 업체인 '밤새' 팀은 영화관 관리 플랫폼 프로토타입을 제공해야함
